3. **Download the Archive**: You will see a file labeled `hytale-dashboard.zip`. Download this file to your computer.

5. **Move to Your Server**: Transfer the extracted folder to your server. You can use SSH or an FTP client for this process.

## ‚öôÔ∏è Running the Dashboard
Once you have transferred the files to your server, follow these steps to run hytale-dashboard:

1. **Open Terminal**: Connect to your server using SSH and open the terminal.
2. **Navigate to the Directory**: Use the `cd` command to go to the folder where you moved the hytale-dashboard. For example:
    ```bash
    cd /path/to/hytale-dashboard
    ```
3. **Install Required Packages**: Ensure you have all necessary packages by running:
    ```bash
    pip install -r requirements.txt
    ```
4. **Start the Application**: Run the following command to start your hytale-dashboard:
    ```bash
    python main.py
    ```
5. **Access the Dashboard**: Open your web browser and go to `http://localhost:8000` to access your dashboard.
